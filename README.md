# Student Management System (Python + SQLite)

This is a simple console-based Student Management System built using Python and SQLite.

## Features
- Add student records
- View all students
- Update student details
- Delete students
- Search student by ID

## Technologies Used
- Python
- SQLite3

## How to Run
1. Make sure Python is installed
2. Clone the repository or download the ZIP
3. Run the program:

```bash
python student_management.py
